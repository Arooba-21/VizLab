# VizLab
Interactive Data Visualization Studio using Streamlit, Plotly, Seaborn &amp; Matplotlib
---

## Project Objective

The goal of VizLab is to provide a **guided, all-in-one platform** for:
- Understanding datasets
- Selecting appropriate visualizations
- Comparing popular Python visualization libraries
- Creating both **interactive dashboards** and **static analytical plots**

## How to Run Locally

1. **Clone the repository**
```bash
git clone https://github.com/Arooba-21/VizLab.git
cd VizLab
````

2. **Install dependencies**

```bash
pip install -r requirements.txt
```

3. **Run the app**

```bash
streamlit run app.py
```
